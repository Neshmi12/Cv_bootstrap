¿Qué Es HTML?

Lenguaje De Marcación Que Sirve Para Definir El Contenido De Las Páginas Web.

Se compone en base a etiquetas, también llamadas marcas o tags, con las cuales conseguimos expresar las partes de un documento, cabecera, cuerpo, encabezados, párrafos, etc.

Etiquetas

Las etiquetas HTML son pequeños bloques de código, que indican al navegador como debe interpretar el contenido recogido entre dichas etiquetas. Cuentan además con atributos que podemos añadir y que en sí, pueden determinar como será el comportamiento específico que tendrá la etiqueta. Comienzan siempre con el símbolo «<» y finalizan con el símbolo «>». entre medias de estos dos símbolos irá el nombre de la etiqueta que queremos que el navegador interprete. Están compuestas por una etiqueta de apertura, y una etiqueta de cierre las etiquetas de cierre se construyen con una barra «/» justo antes de la propia etiqueta y los símbolos «<» y «>».

Esqueleto

Un documento HTML, una página web, contiene un árbol de etiquetas, que tiene una organización básica. la etiqueta raíz de un documento HTML es HTML y dentro tiene dos etiquetas principales:

Head: con la información de cabecera de la página. es información que en principio no aparecerá en el cuerpo de la página, pero que sirve para saber de qué trata la página, cómo deben interpretarla los distintos clientes web, etc.

Body: que es la información del cuerpo, es decir, lo que se verá en el navegador cuando el usuario entre.

Historia de su creación

Creado en 1991 por Tim Berners-Lee, conocido científico a quien se le considera padre de la World Wide Web.

En su momento inicial en el lenguaje HTML se describen una pequeña cantidad de tags o etiquetas que servían para componer documentos.

En 1993 se publicó el primer borrador de especificación del lenguaje HTML basado en SGML.

Formulario

Es una caja de texto que puede contener casillas de verificación, botones y campos alfanuméricos. suelen encontrarse en los sitios web para recolectar los datos personales de los usuarios interesados en la oferta del sitio. Estos datos posteriormente se envían a una base de datos para ser procesados.

Estructura de un formulario HTML y sus elementos.

1. Form: se agregan todos los atributos necesarios para dar estructura a tu formulario

2. input.

3. Select: reduce una lista de valores

4. Text area: representa el espacio donde puede añadir todo el texto que requiera.

5. Fieldset: agrupa varios datos relacionados entre sí mismos

6. Legend: etiqueta o identifica una agrupación de campos.
